GScatter Add-on

About

GScatter is a free scattering add-on build on Blender's geometry nodes. It allows you to scatter any object on a surface and control the look of it by changing the density, scale, rotation and much more. You can use it to quickly create natural environments like meadows or forests.

Installation

Don't unzip this archive. To install GScatter, follow these steps:

  1. Open Blender.
  2. In the top bar, open the Edit menu.
  3. From there, open the Preferences window.
  4. On the left side of the Preferences window, choose the Add-ons tab.
  5. In the top right of the Preferences window, click the Install button.
  6. Browse to the GScatter zip file you downloaded.
  7. In the bottom right of the file browser, click Install Add-on.
  8. You should now see GScatter in the list of add-ons.
  9. Tick the box to the left of the add-on name to enable it.
